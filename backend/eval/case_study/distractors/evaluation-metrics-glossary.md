# Evaluation Metrics Glossary

All RAG evaluation metrics in one place. What is precision at K? The fraction of
top-K results that are relevant. What is the difference between precision at k and
recall at k? Precision is over retrieved, recall is over relevant. What is mean
reciprocal rank? The average inverse rank of the first hit. What is ROUGE score?
An n-gram overlap metric. What is BLEU score used for? Machine-translation-style
overlap scoring. What does faithfulness measure in a RAG answer? Grounding in
context. What reference-free metrics does the RAGAS framework provide? Faithfulness
and answer relevance. This glossary consolidates precision at k, recall at k, MRR,
ROUGE, BLEU, faithfulness, answer relevance, and RAGAS metrics into one summary
page so every evaluation-metric question is answered here in one document.
