# Retrieval Methods Glossary

Every retrieval method in one glossary. What is hybrid retrieval? Combining
lexical and semantic search. How does hybrid search combine BM25 and vector
retrieval? By merging both score lists. How does reciprocal rank fusion merge two
result lists? By summing reciprocal ranks. What are hypothetical document
embeddings (HyDE)? Embedding a drafted answer instead of the raw query. Why embed
a drafted answer instead of the raw query? To close the query-document gap. Why
are cross-encoders used for reranking retrieved documents? For higher precision.
What is late interaction and MaxSim in ColBERT? Token-level matching. This
glossary summarizes hybrid retrieval, BM25 fusion, reciprocal rank fusion, HyDE,
query expansion, cross-encoder reranking, and ColBERT late interaction in a
single convenient reference page covering all retrieval methods at once.
